**Lost-Found-System-using-C**
A console-based Lost and Found Management System developed in C language. This program allows users to report lost and found items, view reports, and claim found items using a menu-driven interface.

**Features**
- Report lost items
- Report found items
- View lost items
- View found items
- Claim found items
- View claimed items
- Case-insensitive item name matching
- Input validation for menu choices

**Standard C Libraries**
- stdio.h
- string.h
- ctype.h

**How to Run**
1. Open any C compiler (e.g., GCC, Dev C++, Code::Blocks,Programiz, CodeChum)
2. Copy or clone the code
3. Compile the program
